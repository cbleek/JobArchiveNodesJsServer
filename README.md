# server-side
Firebase, Express, Node

## Server-side usage(PORT: 8000)
- $ npm i
- $ npm run dev

## Deploy to Heroku
- $ npm i -g heroku
- $ heroku login
- $ heroku create
- $ npm run heroku:add <your-heroku-app-name>
- $ npm run deploy:heroku
